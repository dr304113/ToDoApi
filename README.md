# To Do List API

A Spring Boot, RESTful web API that returns JSON "ToDo" object key/values from MySql Database.

<ul>
<li>GET, POST, DELETE, PUT endpoints verified via Postman.</li>
<li>JDBC for database connectivity</li>
</ul>
